# Karma AI – Scam Baiting Assistant

Karma AI is a web-based AI application designed to interact with scam callers using automated conversational responses in Hindi. The system aims to engage and delay scammers through realistic, context-aware conversations, thereby reducing their efficiency and highlighting the potential of AI in solving real-world problems.

---

## Overview

With the increasing number of scam calls, there is a need for systems that can counter such activities in a practical way. Karma AI explores the idea of using conversational AI to occupy scammers’ time by simulating natural human-like interactions in Hindi. This project focuses on real-time response generation and basic conversational flow handling within a web-based environment.

---

## Features

- Automated AI-driven conversational responses  
- Hindi language support for realistic interaction  
- Web-based interface for user interaction and testing  
- Real-time response handling logic  
- Basic conversational flow management  

---

## Tech Stack

- Backend: Python  
- Frontend: HTML, CSS, JavaScript  
- Libraries: Python libraries for handling response logic and interaction flow  

---

## System Workflow

1. User initiates or simulates a scam interaction through the web interface.  
2. The backend processes the input using predefined logic and response handling.  
3. The AI generates context-based responses in Hindi.  
4. Responses are displayed through the web interface in real time.  
5. The system continues the interaction to maintain engagement.

---

## Key Contributions

---

## Limitations

- Responses are based on basic logic and do not use advanced NLP models  
- Limited contextual understanding compared to production-level AI systems  
- No live call integration (simulation-based interaction)
